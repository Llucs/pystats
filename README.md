# pystats

Estatísticas reais de downloads e informações de pacotes Python do PyPI.

## Funcionalidades

- Pesquisa de pacotes do PyPI
- Estatísticas de downloads (últimas 24h, 7 dias, 30 dias)
- Gráfico de downloads ao longo do tempo
- Informações detalhadas do pacote (autor, licença, versão, dependências)
- Links diretos para o PyPI

## Como usar

Acesse [https://Llucs.github.io/pystats](https://Llucs.github.io/pystats) e digite o nome de qualquer pacote Python na barra de pesquisa.

## Tecnologias

- HTML, CSS e JavaScript puro
- [Chart.js](https://www.chartjs.org/) para gráficos
- [PyPI JSON API](https://pypi.org/help/#json) e [PyPI Stats API](https://pypistats.org/api) para dados
